# Data Visualization App

Приложение для визуализации данных на PyQt5.

## Функционал
- Статистика по данным
- Графики корреляции
- Тепловые карты
- Линейные графики
- Лог действий

## Установка
pip install -r src/requirements.txt
python src/main.py
